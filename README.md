# Beyond the lens into the heart - Photography Portfolio Landing Page

## Nicole Kuria Author


## Project Description
Beyond the lens into the heart is a one-page photography portfolio website built with HTML and CSS.
It showcases favorite photos, includes a biography section, attaches a design plan for review, and provides a contact form for collaboration.

## Features Included
- Introductory quote and paragraph
- Favorite photos section with active links and image previews
- Biography section 
- Design attachment section (`DESIGN.md` and `design-wireframe.svg`)
- Contact form with required input fields
- Custom CSS styling with responsive layout

## Technologies Used
- HTML5
- CSS3

## Project Setup Instructions
1. Clone the repository:
   - `git clone https://github.com/nicolekuria1-create/photography-web.git`
2. Open the project folder in VS Code.
3. Launch `index.html` using Live Server or open it directly in your browser.

## Live Site (GitHub Pages)
Add your deployed link here after publishing:
- `https://nicolekuria1-create.github.io/photography-web/`

## GitHub Repository Setup
1. Create a new repository named `photography-web`.
2. Connect local project to GitHub:
   - `git init`
   - `git add .`
   - `git commit -m "Initial project structure"`
   - `git branch -M main`
   - `git remote add origin https://github.com/nicolekuria1-create/photography-web.git`
   - `git push -u origin main`

## Regular Commit Checklist (Target: 20+ commits)
Use clear commit messages while building, for example:
- `Add semantic HTML layout`
- `Create navigation and hero section`
- `Add favorite photos list with image links`
- `Style biography section and profile card`
- `Attach design wireframe and notes`
- `Improve mobile responsiveness`
- `Refine contact form styling`
- `Update README with deployment steps`

## How to Publish to GitHub Pages
1. Push all latest changes to `main` branch.
2. On GitHub repository page, open **Settings > Pages**.
3. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main` and `/root`
4. Save and wait for deployment.
5. Copy the live URL into the repository description and this README.
