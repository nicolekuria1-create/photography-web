## Design Goal
Create an elegant one-page photography website that feels emotional, clean, and memorable.

## Target Audience
- Photography lovers
- Potential clients
- Instructors/reviewers

## Structure (Planned Before Coding)
1. **Header / Hero**
   - Project title
   - Introductory quote
   - Intro paragraph
   - Navigation links
2. **Favorite Photos Section**
   - Ordered list of selected favorite photos
   - Active links to full image sources
   - Image previews
3. **Biography Section**
   - Website biography
   - Photographer bio text
   - Profile image
4. **Design Review Section**
   - Attached wireframe image for reviews
5. **Contact Section**
   - Name, email, and message form
6. **Footer**
   - Copyright and back-to-top link

## Visual Direction
- Dark cinematic background for a photography mood
- Warm accent color to guide focus
- Serif display font for headings + clean sans-serif body font
- Rounded cards and subtle borders for a polished portfolio look
